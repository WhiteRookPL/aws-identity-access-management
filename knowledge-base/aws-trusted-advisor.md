## Description

An on-line resource to help you reduce cost, increase performance, and improve security by optimizing your AWS environment, Trusted Advisor provides real time guidance to help you provision your resources following AWS best practices.

## Checks

Available for all users, without distinction for support plan:

- Security
- Performance

Access to the six core Trusted Advisor checks to help increase the security and performance of your environment. Checks include:

Security:
- Unrestricted Ports by Security Groups,
- IAM Use,
- MFA on Root Account,
- EBS Public Snapshots,
- RDS Public Snapshots
